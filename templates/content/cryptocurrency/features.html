Our Platforms Protective Features

Your Security is our priority

Organizational Security

CryptoWalletProtection employees must pass a criminal background check as part of the hiring process.

Application Security

We use SQL injection filters and verify the authenticity of POST, PUT, and DELETE requests to prevent CSRF attacks. We rate limit a variety of actions on the site (login attempts, etc). We whitelist attributes on all models to prevent mass-assignment vulnerabilities.

Authentication Security

We protect user data and personal information, including Know-Your-Customer (KYC) information, by encrypting data in storage. Meanwhile, data in transit is secured via end-to-end encryption, ensuring only users have access to their personal information.

Database Security

We hash passwords stored in the database (using bcrypt with a cost factor of 12). We check for strong passwords on account creation and password reset. Application credentials are kept separate from the database and code base.

Real Time Monitoring

Our risk management system analyzes every withdrawal attempt, password reset, two-factor authentication reset and email address change. Unusual activity triggers suspended withdrawals for a minimum of 24-48 hours.

Advanced Data Encryption

We protect user data and personal information, including Know-Your-Customer (KYC) information, by encrypting data in storage. Meanwhile, data in transit is secured via end-to-end encryption, ensuring only users have access to their personal information.

Access Control

Advanced access control provides users with opt-in security features such as IP and wallet address whitelisting, API access control and device management.

Security across the globe

98% of customer funds are stored offline. Offline storage provides an important security measure against theft or loss. We distribute bitcoin geographically in safe deposit boxes and vaults around the world.
